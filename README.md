# 🚀 Tablero de Tareas "Overdrive"

¡Hola! Este es un pequeño pero súper útil tablero digital que armé para organizar las tareas del día a día. Está inspirado en el clásico sistema de mover notitas de papel (post-its) de una columna a otra, pero de una forma mucho más moderna y directamente en tu pantalla.

## ¿De qué se trata?

A veces la cabeza no da para acordarse de todas las cosas que tenemos pendientes. Por eso creé este tablero donde puedes organizar todo en tres columnas muy simples:
- **Por Hacer**: Las cosas que tienes que hacer pero aún no empiezas.
- **En Progreso**: Lo que estás haciendo justo en este momento.
- **Completado**: Lo que ya terminaste (¡la mejor columna de todas!).

## ¿Qué puedes hacer aquí?

*   **Arrastrar y soltar**: Literalmente puedes agarrar una tarea con el ratón y arrastrarla de una columna a otra a medida que vas avanzando con ella. ¡Es muy satisfactorio!
*   **Carpetas separadas**: ¿Tienes tareas de la casa y otras del trabajo? Puedes crear diferentes proyectos para no mezclar tus pendientes.
*   **Detalles de las tareas**: Cuando agregas una nota nueva, puedes ponerle quién es el responsable, qué tan urgente es (alta, media o baja) y hasta asignarle una fecha límite para que nada se te pase.
*   **Se ve genial en todos lados**: Puedes usarlo desde tu computadora o desde el celular, todo se acomoda solo a la pantalla totalmente responsivo. Además, el diseño es oscuro, moderno y súper limpio.

## ¿Cómo se usa?

¡No hay que instalar nada complicado!
1. Solo tienes que buscar el archivo que se llama `index.html` y abrirlo dando doble clic. Se abrirá en tu navegador de internet (Chrome, Safari, el que uses normalmente).
2. Arriba a la derecha verás opciones para crear tu primer Proyecto o Carpeta.
3. Luego, dale al botón **+ Tarea**, anota lo que tengas pendiente y guárdala. 
4. ¡Listo! Ya puedes empezar a mover tus tareas por el tablero.
